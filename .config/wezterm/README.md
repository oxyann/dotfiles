# WerTerm

### 快捷键

|按键|动作|
|:-:|:-:|
|Ctrl+shift+c|复制|
|Ctrl+shift+v|粘贴|
|Ctrl+super+[,]|重命名|
|Ctrl+super+[%]|左右拆分|
|Ctrl+super+["]|上下拆分|
|Ctrl+super+x|关闭当前窗格|
|Ctrl+super+z|最大/小化|
|Ctrl+super+[↑]|向上|
|Ctrl+super+[↓]|向下|
|Ctrl+super+[←]|向左|
|Ctrl+super+[→]|向右|
|Ctrl+super+[num]|跳转指定窗格|
|Ctrl+super+c|新窗格|
